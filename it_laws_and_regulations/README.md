# Informationssäkerhet & IT-juridik
TUC Yrkeshögskola IT-säkerhetsspecialist program

Course module covering Swedish and EU cybersecurity law, compliance frameworks, and legal-technical integration.

## Assignments

### [Jurisdictional Analysis - Swedish Cybersecurity Laws](./jurisdictional-analysis/)
Comparative analysis of three parallel Swedish cybersecurity laws (Dataskyddslagen, Cybersäkerhetslagen, Säkerhetsskyddslagen) with decision framework for incident reporting.

**Key competencies:** Legal analysis, jurisdictional mapping, compliance frameworks, GRC
