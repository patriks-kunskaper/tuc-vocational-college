# TUC Yrkeshögskola - IT-säkerhetsspecialist Portfolio
This repository contains selected coursework from TUC Yrkeshögskola's **IT-säkerhetsspecialist** program (2024-2026), a 2-year vocational higher education focused on IT security specialist competencies.

## About the Program
**Institution**: TUC Yrkeshögskola, Stockholm  
**Program**: IT-säkerhetsspecialist (IT Security Specialist)  
**Duration**: 2 years (400 YH-credits)  
**Focus areas**: Network security, infrastructure security, advanced IT security, information security & IT law, ethical hacking, cloud security, Linux/Unix and Windows security

## Repository Structure
Each project is organized in its own subdirectory with:
- Assignment deliverables (reports, scripts, configurations)
- README explaining context, objectives, and skills demonstrated
- Supporting documentation where relevant

## Key Competencies Demonstrated
- **Security Architecture & Baseline Development**: System hardening, security configuration standards
- **GRC (Governance, Risk, Compliance)**: Risk assessments, compliance frameworks (GDPR, NIS2, ISO 27001)
- **Network & Infrastructure Security**: Firewall configuration, network segmentation, secure system administration
- **Privacy Engineering**: Technical implementation of GDPR requirements, privacy by design
- **Security Automation**: Python, bash, and PowerShell scripting for security tasks
- **IT & Data Law**: Swedish and EU regulatory compliance (GDPR, NIS2 Directive)
